== Description ==

This plugin offers a variety of layouts, making it easy for users to display their content effectively. Available layout types include Grid, List, Carousel, and more—allowing users to choose the style that best suits their needs.

Each layout comes with customizable settings, enabling adjustments to the number of content items, colors, fonts, and other visual elements. The plugin is user-friendly and fully compatible with any website, making it a versatile solution for content presentation.

### Features
- Default layout available.
- Set card border, padding, and border radius.
- Customize card title, description, and button typography, color, padding, and border radius.
- Set card URL and button URL now open in a new tab .

== Installation ==

### From Gutenberg Editor:
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Cards Layout**
3. Click on the **Cards Layout** to add the block

### Download & Upload:
1. Download the **Cards Layout** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`cards-layout.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

### Manually:
1. Download and upload the **Cards Layout** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
